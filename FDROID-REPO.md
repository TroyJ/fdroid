# F-Droid Repo

This repo is the public artifact host for TroyJ Android apps distributed
through the shared F-Droid feed.

## What Lives Here

- published APKs
- F-Droid index files
- app icons
- per-app release metadata

## What Does Not Live Here

- app source code
- shared deployment scripts
- signing secrets

## Publishing Model

- `main` is only for repo metadata
- `gh-pages` is the artifact branch served by GitHub Pages
- the active feed URL is `https://troyj.github.io/fdroid/repo/`
